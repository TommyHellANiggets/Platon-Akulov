ШАГ 1: ЛОКАЛЬНАЯ ПОДГОТОВКА (сейчас, на вашем ПК)
─────────────────────────────────────────────────────
1. Запустите: prepare_deployment.bat
   Или вручную:
   - python manage.py collectstatic --noinput
   - python check_deployment.py

2. Проверьте, что все ✅

ШАГ 2: ОБНОВЛЕНИЕ ФАЙЛОВ
─────────────────────────────────────────────────────
1. Откройте: passenger_wsgi.py
2. Замените USER на ваше имя пользователя:
   SITE_ROOT = '/home/YOUR_USERNAME/public_html'
   VENV_PATH = '/home/YOUR_USERNAME/public_html/venv/lib/python3.10/site-packages'

3. Узнайте версию Python у Reg.ru и обновите путь выше

ШАГ 3: ЗАГРУЗКА НА ХОСТИНГ (через FTP)
─────────────────────────────────────────────────────
1. Откройте FTP клиент (FileZilla, WinSCP и т.д.)
2. Подключитесь: ftp.platon-akulov.ru
3. Загрузите все файлы в: /public_html/
4. НЕ загружайте: venv/, .git/, __pycache__, *.pyc

ШАГ 4: НАСТРОЙКА НА СЕРВЕРЕ (через SSH)
─────────────────────────────────────────────────────
ssh username@platon-akulov.ru
cd ~/public_html
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py collectstatic --noinput
touch tmp/restart.txt

ШАГ 5: ПРОВЕРКА РАБОТЫ
─────────────────────────────────────────────────────
1. Откройте: https://platon-akulov.ru
2. Должна загрузиться главная страница
3. Проверьте админку: https://platon-akulov.ru/admin/
4. Проверьте логи: tail -f ~/public_html/passenger_wsgi.log

═══════════════════════════════════════════════════════════════════════════════

   └─ Она будет создана на сервере автоматически

5. ⚠️  Убедитесь, что .htaccess загружен
   └─ Он нужен для работы Passenger

🚨 РЕШЕНИЕ ЧАСТЫХ ПРОБЛЕМ
═══════════════════════════════════════════════════════════════════════════════

❌ ERROR 500 Internal Server Error
   └─ Проверьте: tail -f ~/public_html/passenger_wsgi.log
   └─ Убедитесь что пути в passenger_wsgi.py правильные

❌ Статические файлы не загружаются (404)
   └─ Запустите: python manage.py collectstatic --noinput
   └─ Проверьте права: chmod 755 collected_static/

❌ ModuleNotFoundError: No module named 'django'
   └─ Активируйте venv: source venv/bin/activate
   └─ Переустановите: pip install -r requirements.txt

❌ БД не найдена (PermissionError)
   └─ Выдайте права: chmod 666 db.sqlite3

❌ Приложение не перезагружается после обновления
   └─ Создайте файл: touch ~/public_html/tmp/restart.txt
